<objective>
Enable reliable utterance segmentation by relaying Deepgram's `UtteranceEnd` events and establish the UI foundation by injecting a Shadow-DOM-isolated panel into the host page.

Purpose: Without `UtteranceEnd` relay, the UI cannot reliably know when a speaker has finished an utterance, leading to merged cards. Shadow DOM ensures host page styles don't break the extension UI.
Output: Relayed events and a visible (but empty) panel in the host tab.
</objective>
